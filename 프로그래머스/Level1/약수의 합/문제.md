# 🧮 약수의 합

> 프로그래머스 - 연습문제 (Level 1) 
> 
> 정답률 `88%`

## 📌 문제 설명

정수 `n`이 주어질 때, `n`의 약수를 모두 더한 값을 return 하도록 함수를 완성하세요.

---

## ✅ 제한 조건

- `n`은 0 이상 3000이하인 정수입니다.

---

## 🧾 입출력 예

| n   | return |
|-----|--------|
| 12  | 28     |
| 5   | 6      |

> 예시 설명:  
> 12의 약수는 1, 2, 3, 4, 6, 12입니다.  
> 이들의 합은 28입니다.
