# 🧮 팩토리얼

> 인프런 Java 알고리즘 문제풀이 연습 문제  
> 정답률 `89%`

---

## 📌 문제 설명

자연수 `N`이 입력되면, `N!`을 구하는 프로그램을 작성하세요.  
`N!`은 `N`부터 1까지의 모든 자연수를 곱한 값입니다.

예를 들어,  
`5! = 5 × 4 × 3 × 2 × 1 = 120`

---

## ✅ 입력

- 첫 번째 줄: 자연수 `N` (1 ≤ N ≤ 12)

---

## ✅ 출력

- 첫 줄에 `N!`의 결과를 출력

---

## 🧾 입출력 예

### 예시 입력 1
> 5

### 예시 출력 1
> 120
