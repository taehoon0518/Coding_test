# 🧮 올바른 괄호

> 인프런 Java 알고리즘 문제풀이 1번  
> 정답률 `30%`

---

## 📌 문제 설명

괄호 문자열이 입력되면 **올바른 괄호인지 판별**하여  
올바른 괄호이면 `"YES"`, 올바르지 않으면 `"NO"`를 출력하는 프로그램을 작성하세요.

- 예를 들어 `(())()` 는 올바른 괄호이지만, `(()()))` 는 올바르지 않은 괄호입니다.

---

## ✅ 입력

- 첫 번째 줄: 괄호 문자열 (최대 길이 30)

---

## ✅ 출력

- 첫 번째 줄: "YES" 또는 "NO"를 출력

---

## 🧾 입출력 예

### 예시 입력 1
> (()(()))(()

### 예시 출력 1
> NO
